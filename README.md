Market Analysis Report (Power BI)

В этом проекте я анализировал рынок и поведение клиентов онлайн-магазина одежды, чтобы помочь компании настроить целевую рекламную кампанию.

Основная цель — понять, как доход клиентов связан с их покупками, выявить группы клиентов с разным уровнем дохода и определить, какие товары им стоит рекламировать.

Что я сделал
Загрузил и подготовил данные из нескольких таблиц (клиенты, покупки, доходы по штатам, инвентарь товаров, демография)
Очистил данные и настроил модель в Power BI
Построил карты, scatter-plots, таблицы и KPI визуализации
Провёл регрессионный анализ, чтобы оценить связь дохода и покупательской активности
Рассчитал прогнозируемый доход клиентов на основе их покупок
Определил, какой продукт логичнее рекламировать каждому клиенту
Сформировал маркетинговую стратегию на основе данных

Основные выводы
Доход и покупательская активность связаны — клиенты с более высоким уровнем дохода совершают больше покупок
Существует заметная взаимосвязь между оценками товара и уровнем возвратов
Построенная модель позволяет оценить доход клиента на основе его последних покупок
Клиенты с низкими тратами — целевая аудитория для недорогих товаров (рубашки)
Кошельковый сегмент — клиенты со средними/высокими покупками → свитера и сумки
